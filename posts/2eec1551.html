第二章：用HTML设置文本

第二节：文本排版
1、段落标签:
<p></p>
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>段落标签</title>
</head>
<body>
<p>第一：影响听力。
经常带着耳机睡觉会导致耳机压迫耳部的神经，不仅会危害耳道的健康，还会刺激耳部神经，导致听力下降，严重的还会导致耳鸣、耳聋，所以大家一定要注意。</p>
<p>第二：压迫耳朵。
长时间使用耳塞听歌，而且喜欢把耳塞塞得很深，耳塞就很容易压迫耳部皮肤，久而久之就容易发生皮肤磨损，一个不小心就容易引发耳道感染，使耳朵发生疾病现象。</p>
</body>
</html>

第五节：框架之间的链接

框架是一种网页布局工具，他的作用是把浏览器是显示空间分为几部分，每个部分都可以独立显示不同的页面

5.1 用cols属性讲窗口分为左右两部分
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>用Cols属性将窗口分为两个部分</title>
</head>
<frameset cols="30%,*">
<frame>
<frame>
</frameset>
</html>
5.2 用rows属性将窗口分为上中下两部分
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>用rows属性将窗口分为上中下两部分</title>
</head>
<frameset rows="30%,10%,*">
<frame>
<frame>
<frame>
</frameset>
</html>
5.3 框架的嵌套
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>框架的嵌套</title>
</head>

<frame>

第八节：总结
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
	</head>
	<body>
		<!-- 表单
		 form 标签
		 action 提交到哪去处理
		 method 提交方式 post  get
		 name 表单名  用于区分表单
		 novalidate :避开验证
		 -->
		 <form action="1.jsp" method="post" name="form1"
		 autocomplete="on">
		 <fieldset>
			 <legend>用户基本信息</legend>
		 
		 <!-- type="text"
		  value:默认值
		  maxlength:最大输入个数
		  size:输入框的长度
		  placeholder:提示信息
		  disabled:未激活
		  required:必填项
		  -->
			 姓名：<input type="text" name="xm" maxlength="5"
			 placeholder="请输入用户名..."/><br>
		<!-- 
			 type="password"  输入内容以密文显示
		-->
			 密码: <input type="password"><br>
			 
		<!-- 单选按纽
		 type="radio" 为了使用单选生效，必须给同个名称
		 checked  默认选中
		 -->
		 性别:<input type="radio" name="gander"> 男
		 <input type="radio" name="gander" checked> 女 <br>
		 <!-- 复选框
		  type="checkbox"
		  checked  默认选中
		  -->
		 爱好: <input type="checkbox">游泳
			   <input type="checkbox" checked>爬山
			   <input type="checkbox">写作业  <br>
			   
		<!-- 文件域 -->
		<input type="file"><br>
		<!-- email类型 
		input type="email"   会作简单的验证
		-->
		邮箱:<input type="email"><br>
		<!-- 电话 
		type="tel"  需要和正则表达式结合使用
		-->
		电话：<input type="tel"><br>
		<!-- URL类型 
		input type="url" 会做简单的验证
		-->
		网址:<input type="url"><br>
		<!-- search类型
		type="search" 输入内容后会在输入后有个删除功能
		 -->
		搜索: <input type="search"><br>
		<!-- color类型 -->
	</fieldset>
		请选择颜色:<input type="color"><br>
		<!-- number类型 
		min:最小值
		max：最大值
		step：步长
		value:默认值
		-->
		请输入数字:<input type="number" min="5" max="100"
		step="5" value="5"/><br>
		<!-- 滑块  range类型
		step:步长
		-->
		<input type="range" step="50"><br>
		<!-- 日期时间类型 -->
		请选择日期时间:<input type="date"><br>
		<input type="time"><br>
		<!-- datalist属性 -->
		<datalist id="list1">
			<option label="百度">https://www.baidu.com</option>
			<option label="谷歌">https://www.google.com</option>
			<option label="京东">https://www.jd.com</option>
		</datalist>
		请选择网址:<input type="text" list="list1"><br>
		<!-- 多行文本框
		textarea  标签
		cols 一行默认显示多少个字节
		rows 默认显示多少行
		 -->
		 备注:<textarea cols="10" rows="3"></textarea><br>
		 <!-- 下拉列表 
		 select  标签
		 multiple  多选
		 selected  默认选中
		 -->
		 省: <select multiple>
				<option value="1">江西</option>
				<option value="2" selected>山东</option>
				<option value="3">山西</option>
			  </select><br>
			  <hr>
		省（市）:<select>
				  <optgroup label="江西">
					  <option>南昌</option>
					  <option>九江</option>
					  <option>新余</option>
				  </optgroup>
				  <optgroup label="山东">
				  	  <option>济南</option>
				  	  <option>烟台</option>
				  	  <option>淄博</option>
				    </optgroup>
			  </select><br><br><br>
		<!-- 提交按纽
		 type="submit"  提交按纽
		 具备提交功能
		 -->
			 <input type="submit"/>
		<!-- 图片按纽
		type="image"
		 具备提交功能
		 -->
			 <input type="image" src="20.jpg" width="35px" height="18px">
		<!-- 普通按纽 
		type="button"  不具备提交功能  和JS结合起来使用
		-->
		<input type="button" value="提交">

		 让页面的信息回到最初
		 -->
